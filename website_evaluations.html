Website Evaluations:

Bad Website: The Big Ugly Website

This website was specifically designed to pretty much break every rule of the CRAP design principles. Overall this may be the worst website ever created, as far as design is concerned.

  • Contrast: The color pairings and text boxes are extremely difficult to read between the poor choices in color and the terrible opacity percentages.
  • Repitition: There is no transcending theme what so ever in this website and each section has an extremely drastic change in style.
  • Alignment: It does not seem to follow any grid consistencies nor does any of the information feel like it is flowing in any particular direction.
  • Proximity: Each element feels clustered and just thrown to the wolves wherever the website automatically put each element.

Good Website: Active Theory

This website may be beyond most websites today in creativity. While it is extremely futuristic, it holds very true to the CRAP fundamental guidelines for what makes a well designed webpage.

  • Contrast: While alot of the room for color is replaced by 3d imaging the text and menu panels follow good color choices to remain clearly visible to the user in the forefront of the graphics.
  • Repitition: As you navigate through the menu the theme is consistent and seem to follow the same trend between each page.
  • Alignment: The alignment of the pages is consistent as you move through the different features of the website.
  • Proximity: The spacing and placement of the elements is neat and clean making each element easy to find and interact with.
